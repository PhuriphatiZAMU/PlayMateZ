## üìå Overview

**PlayMateZ** is a modern web platform designed to solve the problem of lonely gaming. It connects players with professional or casual gaming companions ("PlayMates") to ensure a fun, non-toxic, and engaging gaming experience.

This project focuses on **Front-End Development**, utilizing **Bootstrap 5** for a responsive grid system and **Vanilla JavaScript** for interactive UI components like booking simulations and dynamic content loading.
